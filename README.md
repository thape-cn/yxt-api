
## Installation

Add this line to your application's Gemfile:

```ruby
gem 'yxt-api'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install yxt-api

## Usage

### Config

Create `config/initializers/yxt-api.rb` and put following configurations into it.

```ruby
Yxt.apikey     = 'your_api_key'
Yxt.secretkey  = 'your_secret_key'
```
